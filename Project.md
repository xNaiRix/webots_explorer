# Финальный проект: Автономный робот-разведчик

## Описание

Студент создает автономного робота на базе e-puck в симуляторе Webots, который выполняет миссию разведки: исследует комнату с препятствиями, фиксирует информацию о среде и возвращается на стартовую позицию.


## Сценарий миссии

1. Робот стартует из точки (0, 0).
2. Робот исследует комнату, двигаясь и объезжая препятствия.
3. Через заданное время (или после определенного числа обнаруженных препятствий) робот переходит в режим возврата.
4. Робот поворачивается к стартовой позиции и едет к ней.
5. При достижении стартовой позиции робот останавливается.

## Обязательный минимум


- Код разбит на модули (sensors, navigation, fsm, main).
- FSM содержит минимум 3 состояния (EXPLORE, AVOID, STOP или аналоги).
- Робот объезжает хотя бы 2 препятствия.
- Проект запускается без ошибок.
- Есть README проекта.

## Улучшения для высшего балла

- RETURN_HOME: робот возвращается к (0,0) по одометрии.
- PID для поддержания дистанции при движении вдоль стены.
- Логирование траектории в CSV + визуализация графиком.
- Более умная стратегия исследования (спираль, зигзаг).
- Обнаружение объектов камерой.
- Объезд препятствий при возврате.
- Статистика: количество препятствий, пройденный путь, время миссии.
- Диаграмма состояний FSM в README.
